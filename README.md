# Art of Movement – Yoga Flow

A premium, cinematic, and responsive website for a yoga and meditation studio based in Malta. Built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Responsive Mobile Menu**: Smooth hamburger menu overlay for small screens.
- **Sticky Navigation**: adapts its contrast (dark/glass) based on scroll position.
- **Themed Workshops**: Gallery of upcoming workshops with custom imagery.
- **Smooth Scrolling**: Anchor-based navigation for a seamless user experience.
- **Back to Top**: Dynamic arrow button to return to the hero section.
- **WhatsApp Integration**: Fixed CTA for direct booking queries.

## Tech Stack

- **Framework**: React 18+
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

### Prerequisites

- Node.js (latest LTS recommended)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## How to Push to GitHub

To upload this project to your GitHub account, follow these steps:

1. **Create a new repository** on GitHub (do not initialize with README or license).
2. **Open your terminal** in the project directory.
3. **Initialize Git** (if not already done):
   ```bash
   git init
   git add .
   git commit -m "Initial commit - Yoga site completed"
   ```
4. **Connect to your repository**:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git branch -M main
   ```
5. **Push the code**:
   ```bash
   git push -u origin main
   ```
